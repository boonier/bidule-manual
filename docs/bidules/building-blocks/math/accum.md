# Accum

Accum is a accumulator, or a ramp generator. It's is a swiss army knife, and can be used for a wide range of things.

At its most basic, it is a counter that has a variable rate. It can be used to generate a crude sawtooth wave. It is loopable, which comes in handy for looping samples.
